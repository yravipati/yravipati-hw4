CS1060 HW4 API

This site hosts a serverless API endpoint:

  • POST /api/county_data - Query county health data by ZIP code

Usage Example

curl -H 'content-type: application/json' \
  -d '{"zip":"02138","measure_name":"Adult obesity"}' \
  https://your-site.vercel.app/api/county_data
